# Career Report Admin

大学生职业规划报告管理后台

## 技术栈

- React 18 + TypeScript
- Vite 5
- Ant Design 5
- Zustand
- React Router 6

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 部署

项目配置 GitHub Action 自动部署到 Cloudflare Pages：

1. 推送代码到 `main` 分支
2. GitHub Action 自动构建并部署
3. 访问 Cloudflare Pages 提供的域名
